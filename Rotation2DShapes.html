  • Project Structure same as previous chapter

You have to just update things which are required for rotation

  • Update global variable and in init method init it with identity matrix as we did in last chapter
GLfloat gAngle = 0.0f;
  • Create Update Method which we will use for update of rotation angle
void update(){
    gAngle +=0.5f;
    if(gAngle >= 360.0f)
        gAngle =0.0f;

}
  • In display add rotation code
glm::mat4 rotationMatrix = glm::mat4(1.0);

rotationMatrix = glm::rotate(glm::mat4(1),glm::radians(gAngle),glm::vec3(0.0f,1.0f,0.0f));

modelMatrix *= rotationMatrix;
