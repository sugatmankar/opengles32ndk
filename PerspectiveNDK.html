In previous chapters you got the clear idea how shader code works with orthographics projection 

Lets begin with Perspective projection along with Shader Code handling

  • Create project as stated in previous chapter 'Orthographic Projection with Triangle'
  • Project Structure same as previous chapter

You have to just update things which are required for perspective projection 

  • Update global variable and in init method init it with identity matrix as we did in last chapter
glm::mat4 gPerspectiveProjectionMatrix;
  • In resize method lets make changes required to project Perspective vision
gPerspectiveProjectionMatrix = glm::perspective(45.0f,aspectRation,0.01f,1000.0f);
  • In Init method just change triangle vertices wrt perspective projection range i.e. 0 to 1
const GLfloat triangleVertices[] = {
            0.0f,1.0f,0.0f,//apex
            -1.0f,-1.0f,0.0f,//lb
            1.0f,-1.0f,0.0f//rb
    };
  • In display add translation to see perspectie vision
glm::mat4 translateMatrix = glm::mat4(1.0);

translateMatrix = glm::translate(glm::mat4(1.0),glm::vec3(0.0f,0.0f,-3.0f));

modelMatrix *= translateMatrix;
